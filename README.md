# Daily Data Reporter AI

This project is an automation workflow that automatically collects daily data, generates summaries using AI, and sends reports to the admin.

It also automatically replies to users if their email is available.

---

## 🚀 Features

• Automatic daily trigger (every 24 hours)  
• Collects and filters data automatically  
• Aggregates daily information  
• Generates AI summaries  
• Sends daily report to admin  
• Automatically replies to users via email  

---

## ⚙️ Tools Used

- n8n (workflow automation)
- AI model for summaries
- Email automation
- Data processing nodes

---

## 📊 Workflow Overview

1. Trigger runs every 24 hours  
2. Data is collected and filtered  
3. Information is aggregated  
4. AI generates a summary  
5. Admin receives a report  
6. Users receive automatic email responses
